Angular FlexiPagination directive

Minimal pagination directive supporting client-side and server-side chunking


easy to install
just run bower install angular-flexipagination-directive in your project folder
flexible
it provides both client-side and server-side pagination (see documentation for further details)
efficient
a buffering mechanism allows to minimize server requests when pagination is on server side
